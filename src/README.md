Funcionalidad del código
La idea principal está en que el cliente al introducir el articulo deseado y 
la cantidad, este se guardará. Tras ello, el pedido podra tener el estado de 
RECIBIDO, EN_PREPARACIÓN, LISTO_PARA_ENTREGAR Y ENTREGADO. Por ultimo, se 
calculará el precio a pagar de lo elegido por parte del cliente.
